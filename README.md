## Run Locally

Clone the project

```bash
  git clone https://github.com/SeyBoo/simplyfile-mobile
```

Go to the project directory

```bash
  cd simplyfile-mobile
```

Install dependencies

```bash
  npm run setup
```

Start the server

```bash
  npm run start

  npm run ios/android
```
